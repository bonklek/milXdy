# Tweet PNG User Guide

Tweet PNG exports a reviewed local PNG image from an X/Twitter post share action.

## Where To Find It

- Open X/Twitter.
- Use the post share/action menu when Tweet PNG is enabled.
- Review the generated image before saving or sharing it.

## Common Tasks

- Use Appearance presets to control the export styling.
- Keep Tweet PNG enabled from the Apps Hub or suite settings when you want export actions available.
- If an export looks stale after changing Appearance, refresh the X/Twitter tab and try again.

## Notes

Tweet PNG renders selected post content locally in the browser after a user action. It does not upload the post content to a milXdy service.
