# Remilia Wiki Hyperlinks User Guide

Remilia Wiki Hyperlinks automatically links Remilia ecosystem terms on X/Twitter and shows local hover previews.

## Where To Find It

- Open the extension popup.
- Use the **Wiki** tab for inline link settings.
- Use linked terms directly inside X/Twitter posts.

## Common Tasks

- Turn **Remilia Wiki links** on or off.
- Turn **Wiki previews** on to fetch summaries only when hovering links.
- Adjust **Max links per post** and **Max low-confidence links** when posts feel too busy.
- Change **Link color** to make inline links easier to see.
- Use **Debug mode** only when checking matcher behavior.

## Notes

Inline matching scans visible post text locally. Hover previews fetch Remilia Wiki summaries only when previews are enabled and used.
