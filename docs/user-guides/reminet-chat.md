# RemiNet Chat User Guide

RemiNet Chat provides a docked RemiliaNET chat client and an X Messages integration.

## Where To Find It

- Enable **RemiNet Chat** from the Apps Hub.
- Open **RemiNet Chat** from the side rail.
- On supported X Messages routes, use the **RemiliaNET Chat** entry for the larger chat surface.

## Common Tasks

- Confirm RemiNet login status before opening chat.
- Use the docked panel for recent messages, live updates, reactions, pokes, attachments, media previews, and profile lookups.
- Use minimized mode when you want chat present without taking over the page.
- If chat does not connect, finish RemiliaNET login in the browser, then retry the session and refresh X/Twitter.

## Notes

RemiNet Chat uses the browser RemiliaNET session and connects to RemiliaNET APIs and WebSocket live updates.
